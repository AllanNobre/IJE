## Dependencies

- CMake 3.5.1
- SDL 2
- SDL\_image 2
- SDL\_ttf 2
- SDL\_mixer 2

## How to build

```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./princerescue
```
